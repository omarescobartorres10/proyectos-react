# Proyectos en React

Mini proyectos para practicar React.

## 📁 Proyectos incluidos

| Proyecto | Descripción |
|----------|-------------|
| XFollowCard | Tarjeta de usuario con botón de seguimiento (useState) |
| TicTacToe | Juego de tres en raya con lógica de turnos y detección de ganador |

## 🚀 Tecnologías

- React
- Vite
- CSS Modules

## 🛠️ Cómo correr un proyecto

```bash
cd xfollowcard   # o cd tictactoe
